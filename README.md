# sliver-ui

This template should help get you started developing with Vue 3 in Vite.

## 项目结构

```markdown
.
|--docs // 文档
|--packages
|  |--components // 组件
|  |  |--button // 按钮
|  |  |  |--src 
|  |  |  |  |--button.vue // 组件html
|  |  |  |  |--style.css // 组件样式
|  |  |  |  |--types.ts // 类型
|  |  |  |--__tests__ // 测试
|  |  |  |  |--button.spec.ts // 测试用例
|  |  |--index.ts // 导出组件
|  |--hooks // 钩子
|  |--types // 类型
|  |--play // 测试环境
|  |--sliver-ui // 组件库 用于发布
```
